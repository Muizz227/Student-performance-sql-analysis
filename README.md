# **Student Performance & Attendance Analysis (SQL Project)**

## **Project Objective**
This project analyzes student academic performance and attendance data using SQL to identify trends, top performers, and at-risk students.

## **Dataset Description**
The project uses three tables:
- **students**: student details  
- **subjects**: subject information  
- **marks**: marks and attendance data  

## **Key Analysis Performed**
- **Overall and subject-wise performance analysis**
- **Attendance defaulter identification**
- **Average marks calculation per student**
- **Ranking students based on performance**
- **Attendance impact on academic results**

## **SQL Concepts Used**
- **SELECT, WHERE**
- **JOIN (multi-table)**
- **GROUP BY**
- **Aggregate functions (COUNT, AVG, MIN, MAX)**
- **Window functions (RANK)**
- **Filtering and conditional logic**

## **Tools Used**
- **SQL (MySQL / PostgreSQL / SQLite)**
- **GitHub for version control**

## **Outcome**
This project demonstrates **foundational SQL skills required for a Data Analyst role**, focusing on **real-world analytical thinking rather than basic CRUD operations**.
